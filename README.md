# Stock-Forecasting-ARIMA-Model-Project
Stock Forecaster that employs the ARIMA model on Nvidia stock(or any other stock from Yahoo Finance) from 2023-2025, learning to clean financial data and how to train basic models. Note the project only focused on past data, not real time data.
